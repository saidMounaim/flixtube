## Features:

- Login with GitHub, Google & Magic Email
- Movies, TV Shows & Recently Added Pages
- Add to Watchlist
- Remove from Watchlist

## Technology Stack:

- Next.js
- Next-Auth
- Prisma
- TypeScript
- TailwindCSS
- Shadcn UI

## Usage

### Env Variables

Create a .env file in then root and add the following

```
DATABASE_URL=

GITHUB_ID=
GITHUB_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

EMAIL_SERVER_USER=
EMAIL_SERVER_PASSWORD=
EMAIL_SERVER_HOST
EMAIL_SERVER_PORT=
EMAIL_FROM=
```

## Install Dependencies

```
npm install
```

### Run

```
# Run project
npm run dev
```
